Linear Attention Models

An interactive guide from standard Transformers to S4, Mamba, and DeltaNet — with step-by-step tensor computations.

Why Move Beyond Standard Attention?

In a standard Transformer, self-attention computes:

Standard Attention $$\text{Attention}(Q, K, V) = \text{softmax}\!\left(\frac{QK^\top}{\sqrt{d_k}}\right) V$$

This requires materializing a T × T attention matrix, making it $O(T^2)$ in both time and memory. For a sequence of length $T = 100{,}000$, that matrix has 10 billion entries — far too large.

The goal of linear attention models: Replace the $O(T^2)$ attention with $O(T)$ mechanisms that still capture long-range dependencies. Each model in this guide takes a different approach.

Key Concepts You Need First

State Space Model (SSM)

A system that maps an input sequence $u(t)$ to output $y(t)$ through a hidden state $x(t)$:

$$x'(t) = Ax(t) + Bu(t) \quad \text{(state update)}$$ $$y(t) = Cx(t) + Du(t) \quad \text{(output)}$$

Think of $x$ as a compressed summary of the input history. $A$ controls how the state evolves, $B$ how input enters, $C$ how output is read.

Discretization

Real sequences are discrete (token-by-token), so we discretize with step size $\Delta$:

$$\bar{A} = (I - \tfrac{\Delta}{2}A)^{-1}(I + \tfrac{\Delta}{2}A) \quad \text{(bilinear)}$$ $$\bar{B} = (I - \tfrac{\Delta}{2}A)^{-1} \Delta B$$

This converts the continuous system into a recurrence: $x_k = \bar{A}x_{k-1} + \bar{B}u_k$.

Recurrence vs Convolution

The same SSM can be computed two ways:

$$\textbf{Recurrent:} \quad x_k = \bar{A}x_{k-1} + \bar{B}u_k, \quad y_k = \bar{C}x_k$$
$$\textbf{Convolution:} \quad \bar{K} = (\bar{C}\bar{B},\; \bar{C}\bar{A}\bar{B},\; \ldots,\; \bar{C}\bar{A}^{L-1}\bar{B})$$ $$y = \bar{K} * u \quad \text{(computed via FFT in } O(L\log L) \text{)}$$

Linear Attention

Replace softmax attention with a kernel trick. Instead of $\text{softmax}(QK^\top)V$:

$$y_t = \frac{\sum_{s \le t} \phi(q_t)^\top \phi(k_s) \cdot v_s}{\sum_{s \le t} \phi(q_t)^\top \phi(k_s)}$$

With $\phi = \text{identity}$, this becomes recurrent: maintain state $S_t = S_{t-1} + v_t k_t^\top$ and read $y_t = S_t q_t$. This is $O(T)$ but has no forgetting mechanism.
